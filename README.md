# mail-client

### clonamos el repositorio

git clone https://github.com/Ruben25095/Correo-basico

### Vanos a la ruta del proyecto

cd Correo-basico/

### Instalamos los paquetes necesarios

npm install

### Para iniciar el proyecto 

npm run dev
